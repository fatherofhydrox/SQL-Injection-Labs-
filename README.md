# PortSwigger Web Security Academy Writeups

## Overview

This repository contains my writeups for labs completed on PortSwigger Web Security Academy. The objective of these writeups is to document the testing methodology, vulnerability identification process, exploitation steps, impact analysis, and remediation recommendations for various web application security issues.

Through these labs, I have gained hands-on experience in identifying and exploiting common vulnerabilities found in modern web applications while improving my reporting and documentation skills used in Vulnerability Assessment and Penetration Testing (VAPT).

## Topics Covered

### SQL Injection

* SQL Injection Vulnerability in WHERE Clause Allowing Retrieval of Hidden Data
* SQL Injection Vulnerability Allowing Login Bypass
* SQL Injection Attack Querying Database Type and Version on Oracle

### Cross-Site Scripting (XSS)

* Reflected XSS
* Stored XSS
* DOM-Based XSS

### Cross-Site Request Forgery (CSRF)

* Basic CSRF Vulnerabilities
* CSRF Token Bypass Scenarios

### Broken Access Control

* IDOR (Insecure Direct Object Reference)
* Horizontal Privilege Escalation
* Vertical Privilege Escalation

### Authentication Vulnerabilities

* Weak Password Policies
* Authentication Bypass
* Session Management Issues

### Server-Side Vulnerabilities

* SSRF
* Directory Traversal
* File Upload Vulnerabilities
